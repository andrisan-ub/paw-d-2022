<?php
// Operator Pembanding 
// <, >, <=, >=. ==, != 
var_dump( 3 < 4 ); // true
var_dump( 5 > 2 ); // true
var_dump( 5 <= 3 ); // false
var_dump( 4 >= 4 ); // true
var_dump( 6 == 6 ); // true
var_dump( 3 =! 4 ); //true
?>

Operator Perbandingan

Operator ini berguna untuk membandingkan dan hanya mengecek data tanpa mengecek tipe data nya.

  • < (lebih kecil dari)
  • > (lebih besar dari)
  • <= (lebih besar sama dengan)
  • >= (lebih kecil sama dengan)
  • == (sama dengan)
  • != (tidak sama dengan) 

<?php
var_dump( true && true ); // true
var_dump( true && false ); // false
var_dump( true || false ); // true
var_dump( false || false); // false
var_dump(!false); // true
?>

Operator Logika

  • && 'and' (Jika kedua operand bernilai benar/True)
  • || 'Or' (True jika salah satu)
  • !'Not' (Akan bernilai true jika operand bernilai False) 

<?php
// spaceship operator, memiliki 3 kemungkinan
// 0 jika setara
// -1 jika nilai sebelah kiri lebih kecil
// 1 jika nilai sebelah kiri lebih besar
var_dump(1 <=> 5); // -1
var_dump(1 <=> 1); // 0
var_dump(5 <=> 2); // 1
?>

Spaceship Operator

Sebuah operator yang memiliki 3 kemungkinan return value. Angka yang dihasilkan, antara lain:

  • 0 jika yang dibandingkan setara
  • -1 jika nilai sebelah kiri lebih kecil
  • 1 jika nilai sebelah kiri lebih besar

<?php
// Ouput akan menampikan variabel yang tidak bernilai null 
$num = $value ?? "Ayam Cemani";
echo $num;
// Output akan menampilkan Ayam Cemani
$nam = 800 ?? $value;
echo $nam;
// Output akan menampilkan 800
$nim = "Ayam Cemani" ?? 800;
echo $nim;
// Output akan menampil Ayam Cemani
 $nem = $value ?? $value;
echo $nem;
// output akan error karena diantara keduanya tidak bernilai.
var_dump(10 ?? $d ?? 8); //10
var_dump(20 ?? (8 ?? null)); //20
?>

Null Coalescing Operator (??)

Operator ini akan mereturn value disebelah kiri jika tidak bernilai null, jika null maka mereturn sebelah kanan. Oleh karenanya sistem akan mengouputkan nilai yang tidak null.

Operator Identitas

Operator ini digunakan untuk mengecek sebuah identitas data dengan melihat tipe datanya. 

<?php
// Operator Identitas
// ===, !==, <> 
var_dump(3 == '3'); // akan bernilai true karena tanpa melihat datanya
var_dump(3 === '3'); // akan bernilai false karena ada perbedaan tipe datanya.
var_dump(4 != 3); //true
var_dump(4 !== '3'); //false
var_dump(3 <> 5); // true
?>
