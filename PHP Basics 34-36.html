Operator Arithmetic

Operator Arithmetic / aritmatika adalah operator yang digunakan untuk menghitung operasi matematika dasar. Dari penjumlahan (+), pengurangan (-), perkalian (*), sampai dengan pembagian (/).

$a = 10;
$b = 3;
var_dump($a + $b); // 13
var_dump($a - $b); // 7
var_dump($a * $b); // 30
var_dump($a / $b); // 3.333333...
var_dump($a % $b); // 1 
var_dump($a ** $b); // 1000 (since PHP 5.6) 
var_dump(-$a); // -10

Operator Assignment

Operator Assignment / penugasan adalah operator yang memberikan nilai ke variabel. Nilai dapat berupa nilai tunggal yang sudah jelas atau nilai yang diturunkan dari ekspresi tertentu; misalnya ekspresi aritmatika atau juga ekspresi logika.

$a = 3 + 4 + 5 - 2;
$a = 13;
$a += 14;
$a -= 2;
$a *= 4;
