strlen(), strtolower(), strtoupper(), trim(), concatenate


Fungsi strlen()

Berfungsi untuk mengambil panjang sebuah string. Nilai yang dikembalikan bertipe integer.


$kata = 'Hello';

echo strlen($kata);

//5

Fungsi strtolower()

Fungsi ini digunakan ketika ingin mengubah seluruh karakter pada sebuah variabel menjadi huruf kecil (lowercase).


$kata = 'HELLO';

echo strtolower($kata);

//hello

Contoh:

Fungsi strtoupper()

Fungsi ini digunakan ketika ingin mengubah seluruh karakter pada sebuah variabel menjadi huruf besar (uppercase).


$kata = 'besar';

echo strtoupper($kata);

//BESAR

Contoh:

Fungsi trim()

Fungsi dari trim() yakni untuk menghapus spasi yang berada di depan dan belakang string (spasi yang ada diantara string tidak termasuk).


$kata = ' Tanpa spasi ';

echo trim($kata);

//Tanpa spasi

Contoh:

Operator titik(".")

Operator titik digunakan untuk menggabungkan dua variabel yang berupa string.


$kata1 = 'satu ';

$kata2 ='dua';

$gabung = $kata1.$kata2;

echo $gabung;

//satu dua
