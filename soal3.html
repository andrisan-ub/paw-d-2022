Array

Definisi Array

Array digunakan untuk menyimpan banyak value dalam satu variabel.

Index Array

Untuk mendefinisikan variabel array dapat menggunakan index yang dimulai dari 0

Contoh

<?php
$nama[0] = "Edinson";
$nama[1] = "Wankel";
$nama[2] = "Crapper";
print_r($nama);
?>

Output

Associative Array

Untuk mendefinisikan variabel array dapat menggunakan nama sebagai key.

Contoh

<?php
$creator['Light bulb'] = 'Edison';
$creator['Rotary Engine'] = 'Wankel';
$creator['Toilet] = 'Crapper';
print_r($creator);
?>

Output

Count

Digunakan untuk mengetahui jumlah value pada suatu array.

Contoh

<?php
$family = ["Alvianta", "Rama", "Fika", "Fathiyah"];
$$size = count($family);
echo "size = $size";
?>

Output

Sorting Array

sort() : mengurutkan secara ascending
rsort() : mengurutkan secara descending
asort() : mengurutkan associative array secara ascending berdasar value
ksort() : mengurutkan associative array secara ascending berdasar key
arsort() : mengurutkan associative array secara descending berdasar value
krsort() : mengurutkan associative array secara descending berdasar key

Contoh

<?php
$nama[0] = "Edinson";
$nama[1] = "Wankel";
$nama[2] = "Crapper";
sort($nama);
$nlength = count($nama);
for($i = 0; $i < $nlength; $i++) {
echo $nama[$i];
echo "<br>";};
?>

Output

Menambahkan Value Pada Array

Dalam bahas PHP, Array dapat ditambahkan.

Contoh

<?php
$family = ["Alvianta", "Rama"];
$family[] = "Fika";
$family[] = "Fathiyah";
echo $family[3];
?>

Output

Range

Range dapat digunakan untuk mengisi value pada array yang sesuai dengan kisaran yang dituliskan.

Contoh

<?php
$numbers = range(1,10);
$numlength = count($numbers);
for($i = 0; $i < $numlength; $i++) {
echo $numbers[$i];
echo ", ";}; 
?>

Output

Swapping Value

Array dapat saling bertukar value antar satu sama lain.

Contoh

<?php
$one = 'Alvianta';
$two = 'Rama';
[$one, $two] = [$two, $one];
echo "one = $one";
print("<br>");
echo "two = $two"; 
?>

Output

Destructuring Array

Array dapat men-destructuring yaitu setiap nilai value pada array dapat menjadi value dari sebuah variabel.

Contoh

<?php
$arr = [1, 2, 3];
[$a, $b, $c] = $arr;
echo "$a, $b, $c";
?>

Output
