Arithmetic Operator

Penjelasan

var_dump merupakan fungsi yang digunakan untuk mencetak output pada browser, namun bedanya dengan print maupun echo adalah var_dump akan menampilkan output berupa hasil beserta dengan tipe data hasilnya. Sehingga jika nilai $a adalah int 10 dan $b int 3. Jika digunakan fungsi var_dump maka akan menampilkan tipe data hasil pada outputnya. 

Code
 < ?php 
   $a = 10;
   $b = 3;
var_dump($a + $b);
var_dump($a - $b);
var_dump($a * $b);
var_dump($a / $b);
var_dump($a % $b);
var_dump($a ** $b);
var_dump(-$a);
 ?>

Code
 < ?php
