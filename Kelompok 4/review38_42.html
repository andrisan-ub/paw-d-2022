Comparison Operator

Slide page 38

Penjelasan

Code diatas menggunakan fungsi var_dump dimana outputnya akan berupa tipe data dari output hasil operasi. Pada line 3 hasil operasi adalah true dimana 2 less than 3 sehingga output berupa boolean/bool(true) begitupun pada operasi line selanjutnya. 

Code
 < ?php 
 
   var_dump(2 < 3);
   var_dump(3 < 3);
   var_dump(3 <= 3);
   var_dump(4 <= 3);
   var_dump(2 > 3);
   var_dump(3 >= 3);
   var_dump(3 > 3);
 ?>

Logical Operator

Slide page 39

Penjelasan

Code program menerapkan konsep fungsi var_dump yang digabungkan dengan gerbang logika dimana outputnya akan bertipe data boolean. Pada contoh gambar line 3 output dari konsep gerbang logika AND jika keduanya benar maka output juga benar/TRUE. Contoh selanjutnya adalah pada gambar line 6 yang menggunakan konsep gerbang logika OR dimana jika keduanya salah maka output berupa salah/FALSE. 

Code
 < ?php 
 
 var_dump(true && true);   
 var_dump(true && false);   
 var_dump(true || false);   
 var_dump(false || false);   
 var_dump(!false);  
 ?>

Spaceship Operator

Slide page 40

Penjelasan

Code diatas menggunakan operator spaceship dimana ia akan melihat pada value sebelah kiri dari operator. Jika nilainya < (less than) sebelah kanan maka output berupa angka -1. Jika nilainya > (greater than) sebelah kanan maka output berupa 1. Jika nilai keduanya = (equals to) maka output berupa 0. Fungsi var_dump akan mencetak hasil operasi beserta dengan tipe datanya. Sehingga output berupa tipe data dari hasil operasi diikuti dengan output dari spaceship operator. Pada contoh gambar line 3 misalnya maka program akan mengecek apakah 1 kurang dari 5? jika iya maka cetak -1. Pada gambar line 4 program akan mengecek apakah 1 kurang dari 1? jika sama maka cetak 0. Begitupun pada line selanjutnya. 

Code
 < ?php 
 
   var_dump(1 <=> 5); 
   var_dump(1 <=> 1);     
   var_dump(5 <=> 2);    
    
 ?>

Penjelasan

($_GET['uname']) line 4 merupakan null oleh karena itu nilai dari line 5 yaitu variable $username adalah guest. ($_GET['uname']) line 7 merupakan null, oleh karena itu nilai variable username adalah guest. ($_GET['uname']) line 10 merupakan admin oleh karena itu nilai varible username adalah admin. 

Code
 < ?php 
 
  $uname = isset($_GET['uname']) ? $_GET['uname'] : 'guest';
  var_dump($uname);
    
   $uname = $_GET['uname'] ?? 'guest';

Penjelasan

$a bertipe int, $b bertipe String, $c bertipe int.
Pada contoh gambar line 6 program akan mengecek apakah nilai a dan b sama? iya, jika iya maka print true.
Pada line 7 program akan mengecek apakah nilai $a dan $b sama identik(nilai dan tipe datanya)? tidak, jika tidak print false.
Pada line 8 apakah nilai $a tidak sama dengan $b? tidak, jika tidak maka cetak false. Pada line 9, apakah nilai $a dan $b tidak sama identik(nilai dan tipe data)?iya, maka cetak true.
Pada line 10 apakah nilai $a dan $c sama? iya(sama tipe datanya), maka cetak true.
Pada line 11 apakah nilai $a dan $c tidak sama identik? iya, maka cetak true. 

Code
 < ?php 
  
    $a = 3;
