Looping

Looping PHP

Looping adalah suatu perintah pada PHP yang digunakan untuk mengeksekusi kode yang sama berulang kali selama kondisi tertentu terpenuhi. Terdapat 4 jenis loop berbeda pada PHP, yaitu while, do-while, for, foreach. 

//syntax while loop
while (condition is true){
code to be executed;
}
//syntax do-while loop
do {
code to be executed;
} while (condition is true);
//syntax for loop
for (start; condition; increment) {
code to be executed;
}
//syntax foreach loop
foreach ($array as $value) {
code to be executed;
}
