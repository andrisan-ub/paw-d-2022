Basic PHP Introduction


PHP Variable

Fungsi variable adalah untuk menyimpan sebuah nilai/value tertentu.

Tipe data yang dapat disimpan dalam variable bisa bermacam-macam.

Contoh tipe data Integer, String, Array, dan lain-lain.

Aturan penulisan variable

1. Tipe data variable tidak perlu dideklarasikan

2. Sebelum menuliskan nama variable, perlu diawali dengan notasi dollar " $ "

3. Nama variabel dilarang untuk diawali dengan angka

4. Nama variable harus diawali dengan huruf atau underscore " _ "

5. Variable hanya dapat menyimpan karakter alpha-numeric (A-z, 0-9) dan underscore " _ "

6. Nama variable Case-Sensitive ($var dengan $VAR adalah 2 variable yang berbeda)

Variable_Example
name;
my_name;
NAME;
Your_Name;
TESTING_PHP;
