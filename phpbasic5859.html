unset() digunakan untuk menghapus variabel

$name = “Fred”;

PHP Basics

isset() digunakan untuk mengetahui apakah variabel sudah dibuat atau belum

$s1 = isset($name); // $s1 bernilai false
$name = “Freddy”;
$s2 = isset($name); // $s2 berniali true
