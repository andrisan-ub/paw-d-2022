strlen(), strtolower(), strtoupper(), trim(), concatenate


Fungsi strlen()

Berfungsi untuk mengambil panjang sebuah string. Nilai yang dikembalikan bertipe integer.

Contoh

$kata = 'Hello';

echo strlen($kata);

//5

Fungsi strtolower()

Fungsi ini digunakan untuk mengubah seluruh karakter pada sebuah variabel menjadi huruf kecil (lowercase).

Contoh

$kata = 'HELLO';

echo strtolower($kata);

//hello

Fungsi strtoupper()

Fungsi ini digunakan untuk mengubah seluruh karakter pada sebuah variabel menjadi huruf besar (uppercase).

Contoh

$kata = 'besar';

echo strtoupper($kata);

//BESAR

Fungsi trim()

Fungsi trim() berfungsi untuk menghapus spasi yang berada di depan dan belakang string (spasi yang ada diantara string tidak termasuk).

Contoh

$kata = ' Tanpa spasi ';

echo trim($kata);

//Tanpa spasi

Operator titik(".")

Operator ini digunakan untuk menggabungkan dua variabel yang berupa string.

Contoh

$kata1 = 'satu ';

$kata2 ='dua';

$gabung = $kata1.$kata2;

echo $gabung;

//satu dua
