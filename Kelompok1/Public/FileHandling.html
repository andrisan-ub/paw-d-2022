File Handling

fopen function

fopen() hampir sama seperti readfile(), hanya saja method ini memberikan lebih banyak pilihan/opsi dibandingkan dengan method readfile. Jika kita ingin menggunakan method fopen, terdapat 2 parameter yang harus diisi, yaitu :

fopen (nama file yang dituju, mode file);

Dalam kasus ini, fopen() memiliki mode 'w', artinya file ini akan dibuka untuk ditulis, jika file yang dituju tidak tersedia, maka file tersebut akan secara otomatis dibuat.

readfile function

Fungsi dari readfile() adalah untuk membaca sebuah file dan menuliskannya dalam sebuah output. Perlu diingat bahwa readfile() akan mengembalikan jumlah baris yang terbaca oleh sistem. Jika ingin menggunakan readfile(), kita hanya membutuhkan parameter file yang ingin dibaca, contohnya :

readfile(nama file yang dituju);

Dalam kasus ini, readfile() membaca sebuah file yang bernama files.txt. Di dalam file ini belum diberi isi apapun alias kosong. Angka 0 di output tersebut menandakan jumlah baris yang terisi pada file tersebut karena readfile() tidak hanya mencetak output yang kita tulis, tetapi juga mencetak banyaknya baris yang kita tulis.

fread function

Fungsi dari fread hampir sama seperti readfile, fungsi ini akan membaca sebuah file dan mencetaknya dalam sebuah output. Yang membedakan fread() dengan readfile() adalah fread() tidak akan mengembalikan jumlah baris dari isinya. fread juga memiliki 2 parameter yang harus diisi, yaitu :

fread (variabel "fopen", filesize(nama file));

Dalam kasus ini, fread akan membaca sebuah file yang telah diisi sebelumnya. Dapat dilihat dalam output tersebut, perbedaan antara fread dan readfile terletak pada output tersebut dimana dalam gambar diatas, fread tidak ikut mengembalikan banyaknya baris yang ada pada isinya, hanya mencetak isi dari file tersebut.

fclose function

Fungsi fclose() adalah untuk menutup file yang sudah tidak diproses lagi. Fungsi ini biasanya digunakan jika kita telah menyelesaikan semua hal dalam file tersebut agar tidak memakan resources dengan berlebihan di dalam server. fclose() memiliki 1 parameter yang harus diisi, yaitu :

fclose(nama file yang dituju);

Dalam kasus ini, fclose bertujuan untuk menutup file "files.txt" yang sebelumnya sudah dibuka dan diisi nilainya. Output yang dikeluarkan dari fclose ini tidak ada karena file yang dituju telah ditutup dan jika kita ingin tetap mencetaknya setelah ditutup, maka akan terjadi error.

file_get_contents function

Fungsi dari file_get_contents tidak jauh beda dengan fread dan readfile, sama-sama untuk mencetak output. Yang membedakan adalah method ini hanya membutuhkan parameter file yang dituju dan beberapa pilihan lainnya seperti max_length, start, context, dan include_path. fclose() memiliki beberapa parameter yang harus diisi, yaitu :

fclose(path, include_path, context, start, max_length);

Dalam kasus ini, file_get_contents bertujuan untuk membuka file "files.txt" agar dapat mengakses dan membaca isi dari file tersebut menjadi string. Penggunaan function ini biasanya digunakan jika kita ingin menspesifikan konten file karena menggunakan teknik pemetaan memori untuk meningkatkan kinerja.

fwrite function

Fungsi fwrite adalah untuk menuliskan isi dari file yang dibuka. Fungsi fwrite() berhenti di akhir file atau ketika mencapai panjang yang ditentukan yang diteruskan sebagai parameter, mana saja yang lebih dulu. File, string, dan panjang yang harus ditulis dikirim sebagai parameter ke fungsi fwrite() dan mengembalikan jumlah byte yang ditulis saat berhasil, atau FALSE saat gagal. Adapun syntax dari fwrite sendiri, yaitu:

fwrite(file, string, panjang);

Dalam kasus ini, file fwrite akan menuliskan ulang isi dari file yang dibuka yaitu "files.txt". Lalu setelah "files.txt" berhasil diisi oleh string, selanjutnya dapat dicetak dengan menggunakan fread/readfile untuk mencetak output.

file_put_contents function

Fungsi file_put_contents adalah memeriksa file di mana pengguna ingin menulis dan jika file tidak ada, itu membuat file baru. Jalur file di mana pengguna ingin menulis dan data yang harus ditulis dikirim sebagai parameter ke fungsi dan mengembalikan jumlah byte yang ditulis pada file pada keberhasilan dan FALSE pada kegagalan. Syntax untuk file_put_contents adalah :

file_put_contents($file, $data, $mode, $konteks);

Dalam kasus ini, file_put_contents berfungsi untuk mencetak output yang berada di "files.txt". Isi dari parameter tersebut adalah file yang dibuka dan isinya. Tetapi, yang dicetak adalah panjang string dari isi "files.txt".

fileatime function

Fungsi fileatime() dalam PHP adalah fungsi bawaan yang digunakan untuk mengembalikan waktu akses terakhir dari file tertentu. Fungsi fileatime() mengembalikan waktu akses terakhir file sebagai Unix Timestamp jika berhasil dan False jika gagal. Nama file dilewatkan sebagai parameter ke fungsi fileatime(). Syntax untuk fileatime adalah :

fileatime($filename);

Dalam kasus ini, fileatime akan memberi info waktu akses terakhir dari file "files.txt". Fungsi fileatime() di PHP hanya menerima satu parameter $filename . Ini menentukan file yang waktu akses terakhirnya ingin diperiksa.

filemtime function

Fungsi filemtime() dalam PHP adalah fungsi bawaan yang digunakan untuk mengembalikan waktu terakhir file tertentu saat kontennya dimodifikasi. Fungsi filemtime() mengembalikan terakhir kali file diubah sebagai Unix Timestamp jika berhasil dan False jika gagal. Hasil dari fungsi filemtime() di-cache dan fungsi yang disebut clearstatcache() digunakan untuk mengosongkan cache. Syntax dari filemtime adalah :

filemtime($filename);

Dalam kasus ini, filemtime akan memberikan info mengenai kapan file "files.txt" terakhir dimodifikasi.

fileowner function

Fungsi dari fileowner() adalah mengembalikan ID pengguna (pemilik) dari file yang ditentukan. Kita juga bisa menggunakan syntax "posix_getpwuid()" untuk mengonversi ID pengguna menjadi nama pengguna. Hasil dari fungsi fileowner() di-cache dan fungsi yang disebut clearstatcache() digunakan untuk mengosongkan cache. Syntax dari fileowner adalah :

fileowner($filename);

Dalam kasus ini, fileowner akan memberikan info mengenai ID owner dari file "files.txt".

fileperms function

Fungsi fileperms() dalam PHP adalah fungsi bawaan yang digunakan untuk mengembalikan izin yang diberikan ke file atau direktori. Nama file dari file yang izinnya harus diperiksa dikirim sebagai parameter ke fungsi dan mengembalikan izin yang diberikan ke file dalam bentuk angka jika berhasil dan Salah jika gagal. Syntax dari fileperms adalah :

fileperms($filename);

Dalam kasus ini, fileperms akan mengembalikan izin dengan kode "33206". Fungsi fileperms() di PHP menerima satu parameter $filename . Ini menentukan nama file dari file yang izinnya ingin Anda periksa.

filetype function

Fungsi filetype() dalam PHP adalah fungsi bawaan yang digunakan untuk mengembalikan tipe file dari file atau direktori tertentu. Fungsi filetype() menerima nama file sebagai parameter dan mengembalikan salah satu dari tujuh tipe file jika berhasil dan False jika gagal. Untuk tipe filenya sendiri terdapat 7 tipe file yang berkemungkinan keluar. Syntax dari filetype adalah :

filetype($filename);

Dalam kasus ini, filetype akan mengembalikan tipe file berupa "file" yang berarti itu merupakan file biasa. Ini mengembalikan jenis file pada keberhasilan dan False pada kegagalan.
