Flow Control

Penggunaan if-else 

$nilai = 90;
if ($nilai >= 90) {
	echo 'Sangat Baik';
} else if ($nilai < 90 && $nilai >= 70) {
	echo 'Baik';
} else if ($nilai < 70 && $nilai >= 50) {
	echo 'Sedang';
	echo 'Kurang';
}

If else menggunakan kurung kurawal { }

Membuat pengecekan kondisi terhadap suatu nilai. Jenis if else ini menggunakan kurung kurawal {} dalam mendeklarasikannya 

$nilai = 60;
if ($nilai >= 90):
	echo 'Sangat Baik';
elseif ($nilai < 90 && $nilai >= 70):
	echo 'Baik';
elseif ($nilai < 70 && $nilai >= 50):
	echo 'Sedang';
else:
	echo 'Kurang';
endif;

If else menggunakan colon ( : )

Membuat percabangan alur program untuk mengecek suatu nilai. jenis if else ini menggunakan kolon (:) dalam membuatnya 

$nilai = 80;
echo ($nilai >= 60) ? "Lulus" : "Gagal";

Operator ternary

Membuat operator ternary sebagai metode singkat if else yang memiliki tiga operan, yaitu kondisi awal lalu kondisi benar yang dipisahkan ? dan kondisi salah yang dipisahkan :

Penggunaan switch() 

$nama = "Toni";
switch ($nama) {
	case "Toni":
		echo "Saya Berasal dari Malang";
		break;
	case "Rezky":
		echo "Saya berasal dari Bekasi";
		break;
	case "Ajun" :
		echo "Saya berasal dari Palembang";
		break;
	default : 
		echo "Saya berasal dari Indonesia";
}

Penggunaan Switch Case

Switch digunakan untuk melakukan tindakan yang berbeda berdasarkan kondisi yang berbeda. Switch ini cocok digunakan untuk banyak seleksi kondisi.
