Apa itu AJAX?

Teknik yang digunakan untuk membuat website yang dinamis yang berarti website mampu melakukan update dan menampilkan data terbaru dari server tanpa harus melakukan reload.

Konsep Konvensional

  1. Pemintaan HTTP req dikirimkan dari web browser ke server,
  2. Server menerima dan megambil data,
  3. Server mengirim data yang diminta ke browser,
  4. Web browser menerima data dan mereload halaman untuk menampilkan data. Pada kasus ini, User harus menunggu.

Konsep AJAX

  1. Browser membuat JS call, kemudian mengaktifkan XMLHttpRequest,
  2. Di Background, web browser mengirim request HTTP ke server,
  3. Server menerima, mengambil, dan mengirimkan data ke web browser,
  4. Web browser menerima data yang diminta dan akan ditampilkan di halaman tanpa Reload.

Cara Kerja AJAX

AJAX merupakan sekumpulan teknik pengembangan web yang sistemnya terdiri atas:

  1. HTML/XHTML digunakan sebagai bahasa utama dan CSS untuk menampilkan data,
  2. The Document Object Model(DOM) digunakan untuk menampilkan data yang dinamis dan interaksinya,
  3. XML untuk pertukaran data. Sekarang diganti oleh JSON karena bentuknya mendekati Javascript,
  4. XMLHttpRequest untuk komunikasi tidak langsung (Asynchronous),
  5. Javascript digunakan untukmenyatukan semua teknologi ini.
