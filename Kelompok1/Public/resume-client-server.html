Server and Client-side Scripting

Maksud dari Server-side Scripting adalah bahasa pemrograman web yang pengelolaan datanya dilakukan oleh server. Jadi, setiap web dibuka maka server akan mengirimkan data yang dibutuhkan dari database untuk ditampilkan di web. Sedangkan pada Client-side Scripting pengelolaan datanya dilakukan oleh komputer pengguna. Jadi, ketika web dibuka, komputer milik pengguna akan mendownload script yang bersifat client-side di web tersebut.

                                            <!DOCTYPE html>
                                            <html><body>
                                                    <h1>&l?php
                                                        $hello = "Hello ";
                                                        $world = "world!";
                                                        echo $hello, $world;
                                                    ?></h1>
                                            </body></html>

Halaman 43

Sourcecode diatas merupakan html yang terdapat script php di dalamnya. Penulisan script php harus diawali dengan "<?php" dan ditutup dengan "?>". Menulis variabel dalam php harus diawali dengan tanda $ seperti pada baris ke 4 dan 5, jika tidak diawali dengan tanda $ maka variabel tidak valid. Setelah variabel selesai diinisialisasi kita akan mengambil nilai dari variabel tadi dengan cara menulis perintah “echo” untuk menampilkan nilai variabel pada output, seperti pada baris ke 6 yang tertulis “echo $hello, $world” yang bertujuan untuk menampilkan nilai dari variabel $hello dan $world.


Halaman 47
                                        
                                            <!DOCTYPE html>
                                            <html><body>
                                                <?php
                                                    $var1 = 10;
                                                    $var2 = 20;
                                                    echo “Server output: “, $var1 * $var2;
                                                ?>
                                                <div id=“client”></div>
                                                <script>
                                                    var var1 = 10;
                                                    var var2 = 30;
                                                    var hasil = “Client output: ” + var1 * var2;
                                                    document.querySelector(“#client”).innerHTML = hasil;
                                                </script>
                                            </body></html>

Halaman 47

Pada sourcecode di atas terdiri dari dua bagian, yang pertama menggunakan Server-side Script dan yang kedua menggunakan Client-side Script. Pada bagian pertama, yaitu baris ke 3 sampai dengan 7 merupakan Server-side script, yang diawali dengan menginisialisasi $var1 dan $var2. Kemudian menampilkan nilai dari variabel tersebut dengan memberikan perintah echo “Server output: “, $var1 * $var2;, perkalian variabel ini akan dikerjakan pada server sehingga akan menampilkan output “Server output: 200” ketika program dijalankan. Selanjutnya pada bagian kedua, yaitu pada baris ke 8 sampai 14 merupakan Client-side Script. Pada penulisannya diawali dengan <script>, lalu menginisialisasi kedua variabel var. Lalu untuk mengambil nilai dari variabel menggunakan syntax "var hasil = "Client output: " + var1 * var2;". Lalu pada baris selanjutnya diberi "document.querySelector(“#client”).innerHTML = hasil;" yang berguna untuk mengambil elemen dari HTML dengan id client yaitu hasil.
