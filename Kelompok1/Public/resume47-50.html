Mengelilingi Variable dengan kurung kurawal untuk interpolasi

Error di baris ke 3 code tsb dikarenakan jika tidak menggunakan kurung kurawal maka sistem akan mencari variable “nth”. Jadi fungsi kurung kurawal tersebut untuk memisahkan variable yang kita cari dengan String yang lain. 

<?php
$var = 'world';
$$var = '100';
echo 'Hello $var <br>';
echo "Hello $var <br>";
echo 'Hello $world <br>';
echo "Hello $world <br>";

What’s the output?

Untuk code baris ke 4 dan 6 variable tidak akan di eksekusi sesuai dengan nilai yang sudah dideklarasikan di awal karena echo menggunakan petik satu (‘). Untuk baris ke-5 variable $var akan menampilkan “world” sesuai dengan yang dideklarasikan di awal. Dan untuk baris ke-7 variable $world akan berisi 100 dikarenakan di awal sudah dilakukannya deklarasi nilai world menjadi 100 dengan menggunakan $$(nama_variable)
