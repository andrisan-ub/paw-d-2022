Array

Mendeklarasikan dan Membuat Array 

$anggota[0] = 'Razzaq';//membuat masing" index  
$anggota = array('Bagas', 'Rezky', 'Radit');//menggunakan array()   
$anggota = ['Zulkahfi', 'Ahmad', 'Dafa'];//shorter way

Membuat Indexed Array

Membuat Indexed Array dapat melalui 3 cara, yaitu menambahkan di setiap indexnya, menggunakan array(), dan menggunakan shorter way 

$mahasiswa['Nama'] = 'Bagas Mahda';membuat berdasarkan nama  
$mahasiswa = array('Nama' => 'Bagas Mahda',
                   'NIM' => 215150700111038,); menggunakan array()
$mahasiswa = ['Nama' => 'Bagas Mahda',
              'Prodi' => 'Teknologi Informasi'];//shorter way

Membuat Associative Array

Membuat Associative Array dapat melalui 3 cara, yaitu menambahkan di setiap indexnya, menggunakan array(), dan menggunakan shorter way 

$angka = range(1, 10);
$alfabet = range('a','z');
$urutan_terbalik = range('k','b');

Membuat Array menggunakan function range()

Membuat array dapat menggunakan function range() yang nilainya dapat berupa angka atau alfabet. function range() akan membuat array dari nilai kecil ke besar atau sebaliknya.

Menampilkan Array 

$kelompok1 = ['Razzaq', 'Toni', 'Rezky', 'Adam', 'Bagas', 'Radit'];
echo $kelompok1;//eror
echo $kelompok1[0];

Menampilkan array menggunakan "echo"

Menampilkan array menggunakan "echo" harus disertai indexnya, jika tidak, maka keluaran tampilan webnya akan ada keterangan eror 

$kelompok1 = ['Razzaq', 'Toni', 'Rezky', 'Adam', 'Bagas', 'Radit'];
$mahasiswa = ['Nama' => 'Bagas Mahda',
              'NIM' => 215150700111038,
              'Prodi' => 'Teknologi Informasi'];
print_r($kelompok1);
print_r($mahasiswa);

Menampilkan array menggunakan "print_r()"

Menampilkan array menggunakan "print_r" akan menampilkan kesuluruhan isi dari array. 

$kelompok1 = ['Razzaq', 'Toni', 'Rezky', 'Adam', 'Bagas', 'Radit'];
$mahasiswa = ['Nama' => 'Bagas Mahda',
              'NIM' => 215150700111038,
              'Prodi' => 'Teknologi Informasi'];
var_dump($kelompok1);
var_dump($mahasiswa);

Menampilkan array menggunakan "var_dump"

Menampilkan array menggunakan "var_dump" akan menampilkan kesuluruhan isi array disertai dengan jenis tipe data dan ukurannya.

Menambahkan, Mengkonstruksi, dan Menukar nilai Array 

$kelompok1 = ['Razzaq', 'Toni'];
$kelompok1[] = 'Rezky';
print_r($kelompok1);
['Razzaq', 'Toni', 'Rezky']

Menambahkan nilai pada Array

Menambahkan nilai pada array menggunakan Syntax []. Nilai yang ditambahkan akan berada diakhir index. Penambahan index ini hanya untuk Indexing array 

Indexing Array
$kelompok1 = ['Adam', 'Toni' ,'Bagas'];
[$anggota1, $anggota2] = $kelompok1;
echo $anggota1;echo $anggota2;
Associative Array
$mahasiswa = ['Nama' => 'Bagas Mahda',
              'NIM' => 215150700111038];
['Nama' => $Nama, 'NIM' => $NIM] = $mahasiswa;
echo $Nama; echo $NIM;

Mengkonstruksi nilai array

Menkonstruki nilai array adalah memasukan nilai dari array pada masing-masing variabel. Mengkonstruksi nilai array dapat di Indexing array maupun Associative array. 

$anggota4 = 'Razzaq';
$anggota5 = 'Rezky';
[$anggota4, $anggota5] = [$anggota5, $anggota4];
echo $anggota4;
echo $anggota5;

Menukar nilai menggunakan Array

Menukar nilai pada beberapa variabel dapat menggunakan Array yaitu dengan memasukan variabel pada array. lalu, ditukar menggunakan assignment(=) dengan array yang isinya nilai variabel yang ditukar.

Penggunaan count(), sizeof(), dan foreach 

$kelompok1 = ['Razzaq', 'Toni', 'Rezky', 'Adam', 'Bagas', 'Radit'];
$countKelompok1 = count($kelompok1);
$sizeofKelompok1 = sizeof($kelompok1);
echo $countKelompok1
echo $sizeofKelompok1;

Penggunaan count() dan sizeof()

Penggunaan count() dan sizeof() untuk mengembalikan panjang dari suatu array. 

$kelompok1 = ['Razzaq', 'Toni', 'Rezky', 'Adam', 'Bagas', 'Radit'];
foreach ($kelompok1 as $anggota) {
    echo "Hallo rek, saya $anggota";
}
$mahasiswa = ['Rezky' => 215150700111000,'Radit' => 215150700111001];
foreach ($mahasiswa as $nama => $nim){
    echo "Saya $nama dengan NIM $nim";
}

Penggunaan foreach

Penggunaan foreach ini untuk menampilkan seluruh isi dari Array, baik Indexing Array maupun Associative Array.

Mengurutkan nilai pada array 

$kelompok1 = ['Razzaq', 'Toni', 'Rezky', 'Adam', 'Bagas', 'Radit'];
sort($kelompok1);
print_r($kelompok1);

Penggunaan "sort()"

Penggunaan "sort()" untuk mengurutkan nilai dari atas ke bawah. Penggunaan "sort()" untuk Indexing Array. 

$kelompok1 = ['Razzaq', 'Toni', 'Rezky', 'Adam', 'Bagas', 'Radit'];
asort($kelompok1);
print_r($kelompok1);

Penggunaan "rsort()"

Penggunaan "rsort()" untuk mengurutkan nilai dari bawah ke atas. Penggunaan "rsort" untuk Indexing Array. 

$kelompok1 = ['anggota1' => 'Bagas', 'Anggota2' => 'Radit',
              'anggota3' => 'Toni', 'anggota4' => 'Adam', 
              'anggota5' => 'Rezky', 'anggota6' => 'Razzaq'];
}
rsort($kelompok1);
print_r($kelompok1);

Penggunaan "asort()"

Penggunaan "asort()" untuk mengurutkan nilai dari atas ke bawah berdasarkan nilai di dalamnya. Penggunaan "asort()" untuk Associative Array. 

$$kelompok1 = ['anggota1' => 'Bagas', 'Anggota2' => 'Radit',
              'anggota3' => 'Toni', 'anggota4' => 'Adam', 
              'anggota5' => 'Rezky', 'anggota6' => 'Razzaq'];
}
arsort($kelompok1);
print_r($kelompok1);

Penggunaan "arsort()"

Penggunaan "arsort()" untuk mengurutkan nilai dari bawah ke atas berdasarkan nilai di dalamnya. Penggunaan "arsort()" untuk Associative Array.
