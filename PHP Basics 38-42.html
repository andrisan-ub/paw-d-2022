Operator Comparison

Operator Comparison merupakan operator yang digunakan untuk membandingkan dua buah value (nomor atau string)

Macam-macam Operator Pembanding

  • < (lebih kecil dari)
  • > (lebih besar dari)
  • <= (lebih besar sama dengan)
  • >= (lebih kecil sama dengan)
  • == (sama dengan)
  • != (tidak sama dengan)


var_dump(2 < 3); // true
var_dump(3 < 3); // false
var_dump(3 <= 3); // true
var_dump(4 <= 3); // false
var_dump(2 > 3); // false
var_dump(3 >= 3); // true
var_dump(3 > 3); // false

Operator Logical

Operator logika merupakan operator untuk melakukan operasi logika seperti AND, OR, dan NOT.


var_dump(true && true);
var_dump(true && false);
var_dump(true || false);
var_dump(false || false);
var_dump(!false);

Spaceship Operator

Spaceship Operator atau Three Way Comparison Operator operator Perbandingan yang memiliki 3 kemungkinan return value, yakni:

  • 0 jika perbandingan setara
  • -1 jika nilai di sebelah kiri lebih kecil
  • 1 jika nilai di sebelah kiri lebih besar


var_dump(1 <=> 5);
var_dump(1 <=> 1);
var_dump(5 <=> 2);

Null Coalescing Operator (??)

Operator ini akan mereturn value disebelah kiri jika tidak bernilai null, jika null maka mereturn sebelah kanan. Sistem tidak akan mengeluarkan output Null

$uname = isset($_GET['uname']) ? $_GET['uname'] : 'guest';
var_dump($uname);
 
$uname = $_GET['uname'] ?? 'guest';
 
$_GET['uname'] = 'admin';
$uname = $_GET['uname'] ?? 'guest';
var_dump($uname);

Operator Comparison setara (equal) dan identik (Identical)

Operator ini digunakan untuk membandingkan kesamaan dua buah nilai. Perbedaan perbandingan identical menganggap tipe data dari nilai yang dibandingkannya penting, maka pada perbandingan equal tidak.

  • == (equal)
  • === (identical)
  • != (not equal)
  • <> (not equal)
  • !== (not identical)


$a = 3;
$b = '3';
$c = 5;
var_dump($a == $b);
var_dump($a === $b);
var_dump($a != $b);
var_dump($a !== $b);
var_dump($a == $c);
var_dump($a <> $c);
