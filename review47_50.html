INTERPOLATION

Interpolation adalah proses mengganti nama variabel dalam string dengan nilai-nilai dari variabel tersebut. Untuk menginterpolasi, variabel berada didalam tanda kutip ganda. Cara lain adalah dengan mengurung variabel dengan {}.

Contoh

$n = 12;
echo “You are the $nth person”; // You are the 12th person
echo “You are the {$n}th person”; // You are the 12th person

Untuk mengambil variabel terdapat cara lain yaitu dengan menggunakan tanda petik tunggal ('') atau petik ganda ("")

double dollar ($$x, di mana x adalah nama variabel) adalah variabel referensi yang digunakan untuk menyimpan nilai variabel dolar tunggal ($x) di dalamnya.

Contoh

$var = 'world';
$$var = 100;
echo 'Hello $var'; // Hello
echo "Hello $var""; // $var
echo 'Hello $world'; // Hello world
echo "Hello $world""; // Hello 100

String memiliki index pada setiap karakter seperti array

digunakan untuk mengambil index dari variabel

Contoh

$s = 'Hello';
echo $s[0]; // H
echo $s[-1]; // o

Untuk menguji apakah dua string sama, dapat menggunakan operator ==

Contoh

if ($a == $b) {
    echo “a and b are equal”
}
