PHP Basics

Sebelum mengunggah script php, terlebih dahulu harus memahami struktur penulisan bahasa php. Yaitu, script php diawali dengan <?php dan diakhiri dengan ?> 

<!DOCTYPE html>
<!html><body>
  <h1><?php
    $hello = "Hello";
    $world = "world!";
    echo $hello, $world;
  ?></h1>
</body></html>
