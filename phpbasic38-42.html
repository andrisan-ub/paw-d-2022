PHP Basics

PHP Comparison Operators digunakan untuk membandingkan dua nilai (angka atau string).

PHP Comparison mirip dengan operator di bawah ini.

var_dump(2 < 3); // true
var_dump(3 < 3); // false
var_dump(3 <= 3); // true
var_dump(4 <= 3); // false
var_dump(2 > 3); // false
var_dump(3 >= 3); // true
var_dump(3 > 3); // false

PHP Basics

Begitu juga dengan logical operator seperti di bawah ini.

var_dump(true && true);
var_dump(true && false);
var_dump(true || false);
var_dump(false || false);
var_dump(!false);

PHP Basics

Pada PHP 7, terdapat operator baru, yaitu "spaceship" operator. Spaceship operator mengembalikan bilangan bulat yang kurang dari, sama dengan, atau lebih besar dari nol, bergantung pada apakah $x lebih kecil dari, sama dengan, atau lebih besar dari $y.

var_dump($x <=> $y);

var_dump(1 <=> 5); // -1, karena $x < $y 
var_dump(1 <=> 1); // 0, karena $x = $y
var_dump(5 <=> 2); // 1, karena $x > $y
