PHP Basics

PHP Comparison Operators digunakan untuk membandingkan dua nilai (angka atau string).

PHP Comparison mirip dengan operator di bawah ini.

var_dump(2 < 3); // true
var_dump(3 < 3); // false
var_dump(3 <= 3); // true
var_dump(4 <= 3); // false
var_dump(2 > 3); // false
var_dump(3 >= 3); // true
var_dump(3 > 3); // false


PHP Basics

Begitu juga dengan logical operator seperti di bawah ini.

var_dump(true && true); // true 
var_dump(true && false); // false
var_dump(true || false); // true
var_dump(false || false); // false
var_dump(!false); // true


PHP Basics

Pada PHP 7, terdapat operator baru, yaitu "spaceship" operator. Spaceship operator mengembalikan bilangan bulat yang kurang dari, sama dengan, atau lebih besar dari nol, bergantung pada apakah $x lebih kecil dari, sama dengan, atau lebih besar dari $y.

var_dump($x <=> $y);

var_dump(1 <=> 5); // -1, karena $x < $y 
var_dump(1 <=> 1); // 0, karena $x = $y
var_dump(5 <=> 2); // 1, karena $x > $y


PHP Basics

Pada PHP 7 juga memperkenalkan null coalescing operator. Operator ini mengembalikan nilai $x. Nilai $x adalah expr1 jika expr1 ada, dan bukan NULL. Jika expr1 tidak ada, atau NULL, nilai $x adalah expr2.


          $uname = isset($_GET['uname']) ? $_GET['uname'] : 'guest';
          var_dump($uname); 
          // variable ($_GET['uname']) adalah null, sehingga nilai $username adalah "guest".


          $uname = $_GET['uname'] ?? 'guest';
          var_dump($uname); 
          // variable ($_GET['uname']) adalah null, sehingga nilai $username adalah "guest".


          $_GET['uname'] = 'admin';
          $uname = $_GET['uname'] ?? 'guest';
          var_dump($uname); 
          // variable ($_GET['uname']) adalah admin, sehingga nilai $username adalah "admin".



PHP Basics

Saat membandingkan nilai, kita harus berhati-hati dengan type jugggling. Jenis variabel ditentukan oleh nilai yang disimpannya. Artinya, jika sebuah string dimasukkan ke dalam variabel $var, maka $var bertipe string. Jika setelah itu nilai int ditetapkan ke $var, itu akan bertipe int.

$a = 3;
$b = '3';
$c = 5;

var_dump($a == $b); // true, karena bernilai sama 
var_dump($a === $b); // false, karena tipe data tidak sama 
var_dump($a != $b); // false, karena bernilai sama
var_dump($a !== $b); // true, karena tipe data tidak sama 
var_dump($a == $c); // false, karena tidak bernilai sama  
var_dump($a <> $c); // true, karena tidak bernilai sama
