Client-Server

Sebelum menghubungkan script PHP ke server, harus dipahami terlebih dahulu mengenai aturan penulisan syntax di dalam PHP. Dalam penulisannya, syntax PHP diawali dengan "< ?php>". Bedanya, jika hanya terdiri dari syntax PHP, maka tidak perlu diakhiri dengan “?>”. 

 Contoh :
    < ?php (tanpa spasi)
    $hello = “Hello “;
    $world = “world!”;
    echo $hello, $world;            
    >
