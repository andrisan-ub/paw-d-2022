Hasil tersebut didapat dari kondisi jika input favoritedrink adalah milshake maka akan muncul output It's my favorite drink, tetapi jika input favoritedrink selain milshake maka akan muncul It's not my favorite drink.

Switch

Switch adalah alternatif dari if-else untuk menyatakan lebih dari satu kondisi.

Berbeda dengan if else, perintah pada switch hanya bisa dinyatakan dengan kondisi string atau bilangan (tidak bisa berupa perbandingan, seperti <= atau>)

...
switch ($nilai) {
  case 'A':
    echo 'Nilai anda sangat memuaskan';
    break;
  case 'B':
    echo 'Nilai anda memuaskan';
    break;
  case 'C':
    echo 'Nilai anda cukup';
    break;
  case 'D':
    echo 'Nilai anda kurang';
    break;
}
...

Pada contoh diatas, jika nilai variabel $nilai bernilai 'A', maka akan dijalankan perintah echo 'Nilai anda sangat memuaskan'. Sisa perintah yang ada dibawahnya tidak dijalankan.

Jika variabelnya bernilai 'B', maka akan dijalankan perintah echo 'Nilai anda memuaskan'. Jika bernilai 'C', maka akan dijalankan perintah echo 'Nilai anda cukup', dan jika bernilai 'D', maka akan dijalankan perintah echo 'Nilai anda kurang'.

<?php
$total = 0;
for ($i = 1; $i <= 10; $i++){
$total += $i;};
echo "Total = $total";
?>


Hasil tersebut didapat dari perulangan sebanyak 10 kali yang dideklarasikan nilai awalnya dengan i = 0, jika i <= 10 maka akan terus dilakukan perulangan hingga i> 10. Setiap dilakukan perulangan nilai i akan melakukan increment dan variabel total akan terus ditambah dengan nilai i, total = 1 + 2 + 3 + 4 + ....dst. Sehingga output hasilnya adalah 55.

WHILE

Perulangan While termasuk jenis perulangan uncounted loop atau perulangan yang belum jelas jumlah pengulangannya. Meskipun jumlah perulangan tidak dapat dipastikan, perulangan while tetap dapat dibatasi. Namun, jika ada kesalahan dalam penulisan kode, program akan terus mengulang tanpa henti dan menyebabkan komputer menjadi error.


Contoh, Pertama, kita menentukan variabel $x sama dengan 1. Lalu pada kondisi While, diberikan batasan bahwa variabel x lebih kecil sama dengan 5. Perintah ++ memiliki arti increment atau perulangan akan terus bertambah hingga batas yang telah ditentukan. 

<?php
  $x = 1; 
             
    while($x <= 5) {
    echo "Angka $x <br>";
    $x++;
    }

DO WHILE

Perulangan do... while akan selalu mengeksekusi blok kode satu kali, kemudian akan memeriksa kondisi, dan mengulangi perulangan saat kondisi yang ditentukan benar.


Contoh di bawah ini, pertama-tama menetapkan variabel $x dengan 1 ($x = 1). Kemudian, loop do while akan menulis beberapa output, dan kemudian menambah variabel $x dengan 1 (increment). Kemudian kondisi diperiksa (apakah $x kurang dari, atau sama dengan 5?), dan loop akan terus berjalan selama $x kurang dari, atau sama dengan 5:

<?php
  $x = 1;
    do {
    echo "The number is: $x < br>";
      $x++;
    } while ($x <= 5);

Array

Array digunakan untuk menyimpan banyak value dalam satu variabel.


Index Array

Untuk mendefinisikan variabel array dapat menggunakan index yang dimulai dari 0


<?php
$nama[0] = "Attar";
$nama[1] = "Zidan";
$nama[2] = "Jerry";
print_r($nama);
?>

Associative Array

Untuk mendefinisikan variabel array dapat menggunakan nama sebagai key



<?php
&$creator['Light bulb'] = 'Edison';
&$creator['Rotary Engine'] = 'Wankel';
&print_r($creator);
?>

Count

Digunakan untuk mengetahui jumlah value pada suatu array



<?php
$family = ["Wawan", "Diah", "Rizal", "Rara"];
$$size = count($family);
echo "size = $size";
?>

Sorting Array

rsort() : mengurutkan secara descending
asort() : mengurutkan associative array secara ascending berdasar value
ksort() : mengurutkan associative array secara ascending berdasar key
arsort() : mengurutkan associative array secara descending berdasar value
krsort() : mengurutkan associative array secara descending berdasar key 



<?php
$nama[0] = "Attar";
$nama[1] = "Zidan";
$nama[2] = "Jerry";
sort($nama);
$nlength = count($nama);
for($i = 0; $i < $nlength; $i++) {
echo $nama[$i];
echo "<br>";};
?>

Swapping Value

Array pada php dapat saling bertukar value



<?php
$one = 'Fred';
$two = 'Wilma';
[$one, $two] = [$two, $one];
echo "one = $one";
print("<br>");
echo "two = $two"; 
?>

Destructuring Array

Array pada php dapat didestructuring yaitu setiap value pada array dapat menjadi value dari sebuah variabel 



<?php
$arr = [10, 20, 30];
[$a, $b, $c] = $arr;
echo "$a, $b, $c";
?>
