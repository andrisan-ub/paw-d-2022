PHP Basics

Dalam pemberian nama variabel pada PHP, ada beberapa peraturan yang harus diperhatikan, yaitu :
1. diawali tanda $
2. diawali huruf atau underscore (_)
3. case sensitive (membedakan huruf besar dan kecil)
4. hanya berupa huruf dan angka
5. inisiasi variabel menggunakan tanda sama dengan (=) 

  $paw
    $IniSeptember
    $I_LOVE_PHP
    $_filkom
    $nama = "jang wonyoung";

PHP Basics

Pada source code di bawah ini, variabel yang mengandung 100 adalah variabel $world

$foo = 'bar';
$$foo = 'world';
$$$foo = 100;

Variabel $foo diatas berisi string 'bar', sehingga $$foo adalah variabel $bar, yang berisi string 'world'.

Karena variabel $$foo berisi 'world', maka variabel $$$foo adalah variabel $world, yang berisi 100.
