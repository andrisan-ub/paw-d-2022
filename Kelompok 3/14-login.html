Form Login

File login.html

Ketika user melakukan login, data akan disesuaikan dengan yang ada di "login_process.php" apabila sesuai maka outputnya "Login succeeds" sedangkan jika tidak sesuai maka "Login fails". 


       
                <form action="login_process.php" method="post">
                
                <label>
                Username:
                
                <input type="text" name="username"><br>
                
                </label>
                <br>
                
                <label>
                Password:
                
                <input type="password" name="password"><br>
                
                </label>
                <br>
                <input type="submit" value="Login">
                
                </form>

File login_process.php

Untuk menampilkan data yang dikirimkan, bisa dilakukan echo pada file "login_process.php".


<?php
          $username = $_POST['username'];
          $password = $_POST['password'];
          
          if ($username == 'admin' && $password == '123456') {
          
          echo '<h1>Login succeeds</h1>';
          
          } else {
          
          echo '<h1>Login fails</h1>';
          
          }
          
          ?>
