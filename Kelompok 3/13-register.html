Form register

File register.html

Ketika user mengisi formulir dan mengklik tombol kirim, data formulir akan dikirim untuk diproses ke file PHP bernama "register_process.php". Data formulir dikirim dengan metode HTTP POST.


        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta http-equiv="X-UA-Compatible" content="IE=edge">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Register</title>
        </head>
        <body>
            <h1>Register</h1>
        <form action="register_process.php" method="post">

        <label>Username:

        <input type="text" name="username">
        <label>Password:

        <input type="password" name="password">

        </label>
        <button type="submit" value="Register">Register</button>

        </form>
        </body>
        </html>

File register_process.php

Untuk menampilkan data yang dikirimkan, bisa dilakukan echo pada file "register_process.php".


        </?php

        $username = $_POST["username"];
        $password = $_POST["password"];

        echo "Registering $username with password $password";

        ?>
