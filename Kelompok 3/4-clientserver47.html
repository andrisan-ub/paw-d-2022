Client-Server

Server-side scripts merupakan bahasa pemrograman yang pengolahannya berada langsung di server yang telah terintegrasi dengan web engine. Sedangkan, client-side scripts merupakan bahasa pemrograman yang proses pengolahannya hanya terdapat di browser yang berperan sebagai client.


*Server-side scripts

<?php
    $var1 = 10;
    $var2 = 20;
    echo “Server output: “, $var1 * $var2;
?>

*Client-side scripts

<div id=“client”>
     <script>
         var var1 = 10;
         var var2 = 30;
         var hasil = “Client output: ” + var1 * var2;
         document.querySelector(“#client”).innerHTML = hasil;
     </script>
</div>
