PHP Basics page 52

Fungsi strlen() menghitung panjang string yang diberikan.

Fungsi strlen() mengambil string sebagai argumen dan mengembalikan panjangnya.

Nilai yang dikembalikan adalah tipe size_t (tipe integer yang tidak ditandatangani).

Itu didefinisikan dalam string.


            $s = ‘FILKOM UB’;
            echo strlen($s); // 9


PHP Basics page 53

Fungsi strtolower() digunakan untuk mengubah string menjadi huruf kecil.

Fungsi ini mengambil string sebagai parameter dan mengubah semua huruf besar abjad yang ada dalam string menjadi huruf kecil.

Semua karakter numerik atau karakter khusus lainnya dalam string tetap tidak berubah.

$s = ‘FILKOM UB’; 
echo strtolower($s);
// filkom ub


PHP Basics page 54

Fungsi strtoupper() digunakan untuk mengubah string menjadi huruf besar.

Fungsi ini mengambil string sebagai parameter dan mengubah semua huruf kecil yang ada dalam string menjadi huruf besar.

Semua karakter numerik atau karakter khusus lainnya dalam string tetap tidak berubah.

$s = ‘filkom ub’;
echo strtoupper($s);
// FILKOM UB


PHP Basics page 55

Metode trim() menghapus spasi dari kedua ujung string dan mengembalikan string baru, tanpa mengubah string asli.

Spasi dalam konteks ini adalah semua karakter white space (space, tab, no-break space, dll.) dan semua karakter terminator baris (LF, CR, dll.).

$s = ‘ FILKOM UB ’;
echo trim($s);
// FILKOM UB


PHP Basics page 56

Operator dot adalah operator gabungan PHP.

Dua operan string digabungkan bersama (karakter string tangan kanan ditambahkan ke string tangan kiri) dan mengembalikan string baru.

$s1 = ‘FILKOM ’;
$s2 = ‘UB’;
$s3 = $s1 . $s2;
// FILKOM UB
