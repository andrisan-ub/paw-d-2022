Operator Arithmetic

Operator arithmetic digunakan untuk nilai numerik sehingga menghasilkan suatu value berupa numerik juga.


<?php
$a = 10;
$b = 3;
echo "Pertambahan : " , $a + $b; 
print("<br>");
echo "Pengurangan : " , $a - $b;
print("<br>");
echo "Perkalian   : " , $a * $b;
print("<br>");
echo "Pembaian    : " , $a / $b;
print("<br>");
echo "Modulus     : " , $a % $b;
print("<br>");
echo "Exponent    : " , $a ** $b;
print("<br>");
?>

Operator Assignment

Operator assignment digunakan untuk nilai numerik yang ditambahkan pada variabelnya, variabel yang mendapatkan value adalah variabel sebelah kiri.


<?php
$c = 3 + 4 + 5 - 2;
echo "c = $c";
print("<br>");
$c = 13;
$c += 14;
echo "c += 14 maka c = $c";
print("<br>");
$c -= 2;
echo "c -=2 maka c =  $c";
print("<br>");
$c *= 4;
echo "c *= 4 maka c = $c";
?>
