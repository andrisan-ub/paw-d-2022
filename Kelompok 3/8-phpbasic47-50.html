PHP Basics

Interpolation adalah proses mengganti nama variabel dalam string dengan nilai-nilai dari variabel tersebut. Untuk menginterpolasi, variabel berada didalam tanda kutip ganda. Cara lain dai itu adalah dengan mengurung variabel dengan kurung kurawal

<?php
$n = 12;
echo "You are the $nth person"; // You are the person
echo "You are the {$n}th person"; // You are the 12th person
?>

PHP Basics

untuk mengambil variabel terdapat cara lain juga yaitu dengan tanda petik tunggal ('') atau petik ganda ("")

double dollar ($$x, di mana x adalah nama variabel) dalah variabel referensi yang digunakan untuk menyimpan nilai variabel dolar tunggal ($x) di dalamnya. 

<?php
$var = 'world';
$$var = 100;
echo 'Hello $var<br>'; // Hello $var
echo "Hello $var<br>"; // Hello world
echo 'Hello $world<br>'; // Hello $world
echo "Hello $world<br>"; // Hello 100
?>

PHP Basics

String memiliki index pada setiap karakter seperti array

digunakan untuk mengambil index dari variable

<?php
$s = 'Hello';
echo $s[0]; // H
echo $s[-1]; // o
?>

Untuk menguji apakah dua string sama, dapat menggunakan operator ==

<?php
$a = "nama"

}
