INTRODUCTION


Blade merupakan sebuah template sederhana yang terdapat pada Laravel. File template blade menggunakan ekstensi file .blade.php dan pada umumnya disimpan di direktori resources/views.


Route::get('/', function () {
        return view('greeting', ['name' => 'anya']);
      });

DISPLAYING DATA


#The Loop Variable

untuk mendeklarasikan sebuah foreach loop, tidak perlu menggunakan syntax php, hanya menggunakan fungsi @foreach, kemudian menambahkan kurung kurawal {} untuk membuka dan menutup php echo statement, lalu diakhiri dengan @endforeach.


@foreach ($users as $user)
        @if ($loop->first)
            This is the first iteration.
        @endif
                         
        @if ($loop->last)
            This is the last iteration.
        @endif        
     This is user {{ $user->id }}
     @endforeach

#Conditional Classes

Direktif @class mengkompilasi sebuah string class CSS secara kondisional. Direktif menerima class array di mana array key berisi sebuah class atau atau class baru dengan menggunakan ekspresi boolean untuk mendapatkan nilai.


@php
        $isActive = false;
        $hasError = true;
     @endphp
                         
     <span @class(['p-4', 'font-bold' => $isActive, 'text-gray-500' => ! $isActive, 'bg-red' => $hasError,])></span>
     <span class="p-4 text-gray-500 bg-red"></span>

#Additional Attributes

Direktif @checked dapat digunakan untuk menunjukkan apakah input pada chechbox HTML telah diperiksa. Direktif tersbeut akan mengeluarkan output jika kondisi bernilai "true". 


<input type="checkbox"
        name="active"
        value="active"
     @checked(old('active', $user->active)) />
    
     <select name="version">
        @foreach ($product->versions as $version)
            <option value="{{ $version }}" @selected(old('version') == $version)>
                {{ $version }}
            </option>
        @endforeach
    </select>

#Including Subviews

Direktif @include memungkinkan untuk menyisipkan sebuah blade's view ke blade's view yang lain. Semua variabel yang terdapat pada blade's view parent akan tersedia juga pada blade's view child. Atau dalam istilah lain, biasa disebut inheritance. 


<div>
        @include('shared.errors')
                         
     <form>
        // Konten
     </form>
    </div>

#The @Once Directive

Direktif @once memungkinkan untuk menentukan bagian dari template yang akan dievaluasi dalam sekali rendering. Hal tersbeut berguna untuk melakukan push JavaScript ke header halaman menggunakan stack. 


@once
        @push('scripts')
            <script>
                // KODE
            </script>
        @endpush
     @endonce
                            
     @pushOnce('scripts')
        <script>
            // KODE
        </script>
     @endPushOnce

BUILDING LAYOUTS


#Layout Using Template Inheritance

Layout dapat dibuat dengan template inheritance dimana layout dapat diturunkan ke childnya. Agar child dapat menampilkan template yang diturunkan dari parentnya dapat menggunkana notasi @extends. Notasi @yield digunakan untuk mendefinisikan baris pada parent agar di child dapat disesuaikan. Notasi @section digunakan untuk mendefinisikan baris yang memiliki notasi @yield pada parentnya.


Syntax Parent Layout

<html>
<head>
<title>App Name - @yield('title')</title>
</head>
<body>
@section('sidebar')
This is the master sidebar.
@show
<div class="container">
@yield('content')
</div>
</body>
</html>

Syntax Child Layout

@extends('layouts.app')
@section('title', 'Page Title')
@section('sidebar')
@parent
<p>This is appended to the master sidebar.</p>
@endsection
@section('content')

FORMS


#CSRF Field

CSRF token ini dibuat secara otomatis untuk setiap pengguna. Token ini berupa string acak yang dikelola oleh Laravel untuk memverifikasi permintaan user agar user terlindungi dari serangan CSRF (Cross-Site Request Forgery).


<form method="POST" action="/profile">
@csrf
...
</form>

#Method Field

HTML ini tidak dapat membuat request PUT, PATCH, atau DELETE sehingga perlu untuk menambahkan notasi @method. 


<form action="/foo/bar" method="POST">
@method('PUT')
...
</form>

#Validation Error

Notasi @error berfungsi untuk mengecek suatu atribut apabila terjadi error dapat menampilkan pesan error tersebut. Selain itu juga dapat menambahkan notasi @else yang memberikan tampilan jika tidak error dan diakhiri dengan notasi @enderror


<label for="title">Post Title</label>
<input id="title" type="text" class="@error('title') is-invalid @enderror">
@error('title')
<div class="alert alert-danger">{{ $message }}</div>
@enderror
