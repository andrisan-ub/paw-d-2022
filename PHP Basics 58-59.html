PHP Basics

Perintah unset berfungsi untuk menghilangkan nilai pada suatu variabel

$name = "Fred";
unset($name);
echo $name; // throws a PHP notice

Perintah isset berfungsi untuk memeriksa apakah ariabel sudah ada/terbentuk atau belum

$s1 = isset($name); // $s1 is false
$name = "Fred";
$s2 = isset($name); // $s2 is true
