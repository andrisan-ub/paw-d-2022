?php

Review Slide 38

o;l.[pComparison operator (38) 1. K (lebih kecil) : Bernilai true jika bilangan di sebelah kiri lebih kecil dan false jika bilangan sebelah kiri lebih besar. Pada, contoh 2 kurang dari 3 bernilai true karena 2 lebih kecil dari 3 dan 3 kurang dari3 bernilai false karena 3 sama dengan 3.
2. > (lebih besar) : Bernilai true jika bilangan di sebelah kiri lebih besar dan false jika bilangan sebelah kiri lebih kecil. Pada contoh, 2>3 bernilai false karena seharusnya 2 lebih kecil dari 3 dan 3> 3 bernilai false karena 3 sama dengan 3.
3. K= (kurang dari sama dengan) : Bernilai true jika bilangan di sebelah kiri lebih kecil atau sama dengan bilangan di sebelah kanan dan false jika bilangan sebelah kiri lebih besar. Pada contoh, 3 kurang dari sama dengan 3 bernilai true karen 3 sama dengan 3 dan 4 kurang dari sama dengan 3 bernilai false karena 4 lebih besar dan sama dengan 3.
4. >= (lebih dari sama dengan) : Bernilai true jika bilangan di sebelah kiri lebih besar atau sama dengan bilangan di sebelah kanan dan bernilai false jika bilangan sebelah kiri lebih kecil. Pada contoh, 3>=3 bernilai true karena 3 sama dengan 3.

Review Slide 39

Logical operator (39) 1. && (AND) : Bernilai true jika kedua variabel bernilai true atau false. Bernilai false ketika salah satu variabel bernilai true atau false.
Contoh: true && true //true
true && false //false
2 || (OR) : Bernilai true jika kedua variabel bernilai true atau salah satu bernilai true. Bernilai false jika kedua variabel bernilai false.
Contoh:
true || false //true
false || false //false
3. ! (NOT) : Membuat suatu variabel bernilai salah atau kebalikan dari kondisi sebenarnya.
Contoh :
!false //true

Review Slide 40

Spaceship Operator(40)
=> (Spaceship) : Operator baru yang ada pada PHP 7. Memiliki kemungkinan 3 nilai:
a. 0 jika yang dibandingkan setara
b. 1 jika nilai sebelah kiri lebih besar
c. -1 jika nilai sebelah kiri lebih kecil
Contoh:
var_dump (1-=>5); // -1
var_dump (1-=>1); // 0
var_dump (5-=>2); // 1

Interpolasi adalah proses mengganti nama variabel dalam string dengan nilai-nilai dari variabel tersebut
$n = 12;
echo "You are the $nth person";
echo "You are the {$n}th person";
Kode di baris ke dua akan menghasilkan output "You are the person" dan tidak menampilkan $n karena tidak diinterpolasi.
Kode di baris terakhir akan menghasilkan output "You are the 12th person" karena terdapat interpolasi ditandai dengan "{}" yang diisi dengan $n, sehingga nilai pada variabel tersebut dimasukan ke echo.
