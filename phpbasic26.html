PHP Basics

Pada source code di bawah ini, variabel yang mengandung 100 adalah variabel $world

$foo = 'bar';
$$foo = 'world';
$$$foo = 100;

Variabel $foo diatas berisi string 'bar', sehingga $$foo adalah variabel $bar, yang berisi string 'world'.

Karena variabel $$foo berisi 'world', maka variabel $$$foo adalah variabel $world, yang berisi 100.
