Operator Arithmetic

Operator arithmetic digunakan untuk nilai numerik sehingga menghasilkan suatu value berupa numerik juga.


$a = 10;
$b = 3;
var_dump($a + $b); // 13         --> pertambahan 
var_dump($a - $b); // 7          --> pengurangan 
var_dump($a * $b); // 30         --> perkalian 
var_dump($a / $b); // 3.33333... --> pembagian 
var_dump($a % $b); // 1          --> modulus 
var_dump($ ** $b); // 1000       --> eksponen 
var_dump(-$a); // -10            --> negatif

Operator Assignment

Operator assignment digunakan untuk nilai numerik yang ditambahkan pada variabelnya, variabel yang mendapatkan value adalah variabel sebelah kiri.


$c = 3 + 4 + 5 * 2; // 17 
$d = 13; 
$d += 1; // 14   --> value akan masuk pada variabel yang kiri
$d -= 2; // 12 
$d *= 4; // 48
