Server- and Client-side Scripting

Dalam bahasa pemrograman PHP, istilah Server-side digunakan dalam penerapan eksekusi script/code program yang dilakukan oleh web server pengguna. Folder directory yang ditetapkan untuk menyimpan script dari program adalah htdocs atau www dengan tipe file .php. Untuk membuka atau melihat output hasil eksekusi program, maka pengguna perlu membuka browser dan mengetikkan localhost/namafolder/namafile pada search bar broser. Pengguna juga dapat menjalankan script program pada software yang digunakan untuk membuat program dalam bahasa PHP. 

<!DOCTYPE html>
<html><body>
            
<h1><?php
$hello = "Hello";
$world = "world";
echo $hello, $world;
?><h1>
<html><body>

Contoh Penggunaan Server-side pada PHP

Pada gambar diatas, terlihat bahwa program yang digunakan untuk menjalankan code PHP dapat dieksekusi dan menampilkan output pada halaman browser pengguna dengan kode URL sesuai nama folder dan nama file dari script yang ingin dieksekusi.
