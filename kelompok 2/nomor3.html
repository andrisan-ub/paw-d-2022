Penerapan Looping

Loop digunakan untuk mengeksekusi blok kode yang sama berulang kali, selama kondisi tertentu benar. Di PHP, terdapat beberapa jenis loop, yaitu:
- while: loop melalui blok kode selama kondisi yang ditentukan benar
- do... while: loop melalui blok kode sekali, dan kemudian mengulangi loop selama kondisi yang ditentukan benar
- for: loop melalui blok kode beberapa kali
- foreach: loop melalui blok kode untuk setiap elemen dalam array
Berdasarkan contoh di atas, telah dibuat dua macam looping php yang menggunakan dua jenis loop, yakni for loop dan foreach loop array:
a. Pada contoh for loop, dibuat variabel x dengan nilai 1 dan x kurang dari sama dengan 10 dengan x increment. Output yang dihasilkan berupa teks antrian dengan nomor yang terus bertambah mulai dari angka satu sampai dengan angka 10.
b. Padabcontoh foreach loop dalam array, dibuat array dengan variabel mahasiswa dan nilai "Jasmine", "Zoro", dan "Luffy". Kemudian dibuat looping foreach variabel mahasiswa sebagai variabel nama. Output yang akan dihasilkan berupa teks menyapa, yakni "Halo" dengan diikuti nama dari setiap nilai dalam variabel mahasiswa seperti pada gambar di bawah.

Array

Array dalam PHP ada 3 jenis
Kita dapat mengidentifikasikan array berdasarkan posisinya
1. Indexed Array merupakan suatu array dengan index numeric dimana value atau nilainya disimpan secara liniear (GAMBAR ARRAY 1)
2. Associative Array merupakan suatu array dengan index string dimana value nya tidak disimpan secara linear, melainkan dengan memberikan key tertentu (GAMBAR ARRAY 2)
3. Multidimensional Array merupakan array yang berisi array tunggal atau banyak di dalamnya dan dapat diakses melalui beberapa indeks
