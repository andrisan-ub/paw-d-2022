Kode pengenalan pada PHP di materi Client-Server slide 47

Client-side scripts ialah bahasa pemrograman yang dalam proses mengelolanya hanya terdapat di browser yang memiliki peran sebagai client.
Dari code yang diberikan, [echo "server output: ", $var1 * $var2;] di mana hasilnya akan menjadi Server output: 200 karena variabel $var1 memiliki nilai 10 dan variabel $var2 memiliki niai 20 dan dari perintah echo memiliki perintah perkalian sehingga, output yang ke luar adalah 200.

var1 = 10;
var2 = 20;
echo "Server output: ", $var1 * $var2;
var var1 = 10;
var var2 = 30;
var hasil = "Client output: " + var1 * var2;
document.querySelector("#client").innerHTML = hasil;
