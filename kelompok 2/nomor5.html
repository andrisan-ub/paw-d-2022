Form Registrasi

Membuat form registrasi dengan syntax berikut: 

<html>
<h1>Register</h1>
<form action="register_process.php" method="post">
<label>Username:
<input type="text" name="username"></label>
<label>Password:
                
<input type="password" name="password">
</label>
<label>Password:
                
<input type="password" name="password">
</label>
<input type="submit" value="Register">
</form>
</html>

Syntax di atas disimpan dengan nama "register.php"

Langkah selanjutnya

Membuat file "register_process.php" dengan isi seperti berikut: 

<?php
§username = §_POST['username'];
§password = §_POST['password'];
echo "Registering §username with password §
password";
?>

Web browser akan menampilkan form registrasi dari file "register.php" dengan 2 field yaitu
                    
Username dan Password. Ketika sudah mengisi kedua field tersebut dan memencet tombol "register", 
                    
maka kita akan diarahkan ke tampilan file "register_process.php" dan diperlihatkan pesan jika berhasil register. 
                    
Peralihan halaman dapat terjadi karena terdapat atribut "action" dengan nilai "register_process.php" yang akan berfungsi 
                    
ketika pengguna memencet tombol "register" dan nilai-nilai yang diinputkan di halaman sebelumnya akan ikut ditampilkan di halaman 
                    
baru berkat adanya variabel $username = $_POST['username']; $password = $_POST['password'];

Ajax Chat

AJAX (Asynchronous Javascript And XML) merupakan sebuah teknik interaktif yang digunakan untuk membuat aplikasi web.
Aplikasi web dapat mengambil data dari server secara Asynchronousdi belakang tanpa perlu merefresh halaman.
AJAX menggunakan asynchronous data transfer (HTTP request) antara browser dan web server, yang memperbolehkan halaman web memanggil bit yang kecil atau seluruh informasi dari server. Teknik AJAX membuat aplikasi web menjadi kecil, cepat dan lebih user-friendly.
