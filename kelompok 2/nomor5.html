File Handling

Fungsi fopen() adalah fungsi yang digunakan untuk membuka file atau URL. Jika fungsi ini gagal, ia akan mengembalikan FALSE dan kesalahan yang ada pada kegagalan program PHP. Fungsi ini bekerja dengan dua parameter. Pada parameter pertama, kita memasukkan path file yang akan kita akses, dan pada parameter kedua, kita memasuki mode yang akan menunjukkan untuk apa kita mengakses file tersebut. 

Jika path URL diawali dengan 
              http:// maka koneksi HTTP akan dimunculkan.
              Jika path URL diawali dengan 
              ftp:// maka koneksi FTP akan dimunculkan.

File Read

Fungsi fread() ini memungkinkan Anda membaca file PHP, tetapi hanya menampilkan jumlah byte maksimum yang dipilih dalam file yang dibuka. 

Contoh:
          
<?php
            $file_handle = fopen("contoh.txt", "r");
            $contents = fread($file_handle, 1024);  
          ?>

File Size

Fungsi filesize() digunakan untuk mengembalikan ukuran file.

File Exist

Pastikan untuk selalu memeriksa eksistensi file sebelum membaca isi didalamnya.

File Get Content dan File Put Content

Ada beberapa fungsi shortcut untuk membaca sebuah file yaitu menggunakan file get contents. Fungsi ini bisa digunakan untuk membaca file menjadi string. 

Contoh:
<?php
            echo file_get_contents("contoh.txt", "a string"); 
            ?>
Atau:
<?php
              echo file_put_contents("contoh.txt");
              ?>

File Write

Fungsi ini dapat digunakan untuk menulis ke file yang terbuka. 

Fungsi ini akan berhenti di akhir file (EOF)
            atau saat mencapai panjang yang ditentukan,
            mana saja yang lebih dulu.

File Close

File PHP ditutup menggunakan fungsi fclose(). 

Contoh:
              
<?php
                $file = fopen("contoh.txt", "r");
                fclose($file);
              ?>

Ajax Chat

AJAX (Asynchronous Javascript And XML) merupakan sebuah teknik interaktif yang digunakan untuk membuat aplikasi web.
Aplikasi web dapat mengambil data dari server secara Asynchronousdi belakang tanpa perlu merefresh halaman.
